MSP360 is very excited to announce the release of Standalone CloudBerry Backup version 7.1! This new release introduces many new features and fixes that our customers will love. However, the latest feature is the long-awaited Grandfather-Father-Son retention policy.

Grandfather-Father-Son, or GFS for short, is a backup retention policy that is based on storing several full backups. Weekly (son), monthly (father), and yearly (grandfather).

This backup retention feature also provides an additional level of security and allows you to create a flexible archival strategy to optimize the number of full backups at the same time.

Read this blog post for more details on the release. It explains the concept of GFS, reviews how it works in CloudBerry Backup, and guides you through the backup plan setup process.
